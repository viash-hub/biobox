Usage: bwa samse [-n max_occ] [-f out.sam] [-r RG_line] <prefix> <in.sai> <in.fq>

Options:
       -n INT     maximum occurrences for read mapping [varies by BWA version]
       -f FILE    output SAM file [stdout]
       -r STR     read group header line such as '@RG\tID:foo\tSM:bar'

Arguments:
       <prefix>   BWA index base name (prefix of .amb, .ann, .bwt, .pac, .sa files)
       <in.sai>   input SAI file generated by BWA aln
       <in.fq>    input FASTQ file with single-end reads

Notes: 
- BWA samse generates SAM format alignments for single-end reads
- Input SAI file must be generated first using BWA aln
- For paired-end reads, use BWA sampe instead
- Part of the classic BWA pipeline: bwa index -> bwa aln -> bwa samse
